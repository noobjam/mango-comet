# Existing design system

The viewer uses a dark geospatial operations aesthetic. Preserve the existing CSS tokens in `server/static/styles.css`:

- Background: `--bg: #07100e`
- Primary text: `--ink: #f5faf7`
- Accent: `--accent: #73e2b4`
- Surfaces: `--surface`, `--surface-raised`, `--surface-soft`
- Borders: `--line`, `--line-strong`
- Secondary text: `--muted`, `--muted-strong`
- Status: `--warning`, `--danger`
- Shape/elevation: `--radius`, `--shadow`

Map colors must represent a small reviewed family taxonomy, not every learned internal archetype. Lifecycle/current severity is encoded independently from family identity. Novel or unassigned events use a neutral grey treatment.
